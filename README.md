## Возможности

Программа позволяет:

- создавать ориентированные и неориентированные взвешенные графы;
- добавлять и удалять вершины и рёбра;
- перемещать вершины мышью на рабочем поле;
- сохранять граф в JSON и загружать его обратно;
- выполнять BFS и DFS;
- находить кратчайшие пути алгоритмом Дейкстры;
- строить минимальное остовное дерево алгоритмом Краскала;
- просматривать шаги алгоритмов вручную или в автоматическом режиме;
- видеть подсветку текущих вершин и рёбер.

## Технологии

- Python 3.10+;
- Tkinter;
- Pytest;
- JSON;
- Docker.

## Запуск на Windows

Самый простой способ — открыть файл `START_ПРОГРАММЫ.bat` двойным щелчком.

Либо выполнить из Git Bash:

```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
python app.py
```

## Тесты

```bash
pytest
```

Или двойным щелчком открыть `ЗАПУСТИТЬ_ТЕСТЫ.bat`.

## Docker

```bash
docker build -t graph-visualizer-a03 .
docker run --rm graph-visualizer-a03
```

Запуск тестов в контейнере:

```bash
docker run --rm graph-visualizer-a03 pytest
```
