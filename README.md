# Визуализатор алгоритмов на графах

Учебный проект по ознакомительной практике. Вариант **А-03**.

**Студент:** Василенко Денис Константинович  
**Группа:** БИС-24-2  
**Направление:** Информационные системы и технологии

## Цель

Разработать настольное приложение, которое позволяет работать с графом и
визуализировать алгоритмы обхода и поиска:

- обход в ширину (BFS);
- обход в глубину (DFS);
- алгоритм Дейкстры;
- минимальное остовное дерево алгоритмом Краскала.

## Текущий этап

Добавлена модель графа: вершины, взвешенные рёбра, ориентированный и неориентированный режимы.

## Запуск

На Windows можно дважды нажать `START_ПРОГРАММЫ.bat`.

Из Git Bash:

```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
python app.py
```
